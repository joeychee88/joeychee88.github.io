This application requires both frontend and backend services. GitHub Pages only serves static files. Please deploy using one of the options below.

Quick Start Options

Option 1: Vercel (5 minutes)

git clone https://github.com/joeychee88/joeychee88.github.io.git
cd joeychee88.github.io/kult-planning-engine
npm install -g vercel
cd backend && vercel
cd ../frontend && vercel

Option 2: Local Development (2 minutes)

git clone https://github.com/joeychee88/joeychee88.github.io.git
cd joeychee88.github.io/kult-planning-engine/backend
npm install && npm start
# New terminal:
cd ../frontend && npm install && npm run dev
