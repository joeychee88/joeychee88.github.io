## Key Differences from Manual Selection

### Manual Persona Selection (Select Audiences Tab)
- User browses ~50+ personas
- Manually checks boxes
- Must remember which personas work for their objective
- Time consuming for repeat campaigns
- No historical reference

### Saved Audience Groups (Target Segment Tab)
- Pre-configured persona combinations
- Named and organized by campaign
- One-click load
- Includes metadata (reach, industry, objective)
- Reusable across campaigns
- Historical tracking (usage count, dates)

## Benefits
1. **Speed**: 1 click vs 5+ clicks
2. **Consistency**: Same personas for similar campaigns
3. **Knowledge Transfer**: Team members can see successful combinations
4. **Data Driven**: Usage count shows popular segments
5. **Efficiency**: No need to remember personas for repeat objectives
